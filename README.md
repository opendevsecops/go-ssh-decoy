# SSH Decoy

## Usage

```sh
Usage of /bin/ssh-decoy:
  -host string
    	Host address to listen to (default "0.0.0.0")
  -port int
    	Port number to listen to (default 2022)
  -priv string
    	Private key file
  -size int
    	Private key size (default 2048)
```
